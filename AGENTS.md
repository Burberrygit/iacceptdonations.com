# iacceptdonations.com Creative Direction

## Site Concept

The site is a one-page joke donation website for `iacceptdonations.com`.
Its only real purpose is to accept donations.

## Core Joke

A glossy fake-startup donation page whose entire business model is:

> we disrupted revenue by simply asking for it

That exact line should be the main page header.

## Tone

Corporate parody, crowdfunding satire, absurdly confident, and shamelessly transparent.
It should feel like an homage to the "Go Fund Yourself" style of startup/crowdfunding parody without using South Park names, direct quotes, characters, art, or assets.

## Page Ideas

- Main headline: "we disrupted revenue by simply asking for it"
- Brand signal: `iAcceptDonations.com`
- Donation-first layout with preset amounts and a custom amount field.
- Donation actions should support live Stripe and PayPal payment links through `script.js`.
- Hosting plan: GitHub Pages for the static site, Cloudflare proxy/protection, and a Cloudflare Worker at `/api/create-checkout-session` for exact-amount Stripe Checkout without paid hosting.
- Copy should sound polished and overconfident while admitting the whole model is receiving money.
- Fake startup metrics are encouraged:
  - Revenue model: Yes
  - User acquisition: You
  - Burn rate: Emotionally sustainable
  - Exit strategy: Refresh button
- Donation buttons can update a fake gratitude/status message.

## Visual Direction

Modern tech startup / fintech landing page with polished product visuals, soft shadows,
clean spacing, and crisp conversion-focused donation controls.
No South Park assets, no cartoon imitation, and no copyrighted imagery.
The joke should come from the copy and the absurd confidence of the presentation.
